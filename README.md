# Calendar Events

Flutter calendar and event app built with BLoC and Dio/Retrofit.

## Overview

This project demonstrates a calendar-focused mobile screen with structured data flow and a layered application architecture.

## Features

- Calendar screen for browsing dates and events
- Syncfusion DatePicker integration
- BLoC-based state management
- Dio and Retrofit for networking
- Loader overlay and error snackbar handling
- Data, domain, presentation, and server layers

## Stack

- Flutter
- Dart
- flutter_bloc
- Dio
- Retrofit
- Freezed
- json_serializable
- Syncfusion DatePicker
- loader_overlay
